## Tech Stack

- **Backend**: Node.js, Express, MongoDB, Redis
- **Frontend**: React.js, Tailwind CSS
- **Blockchain Integration**: Web3.js, Ethereum JSON-RPC
- **DevOps**: Docker, NGINX, AWS EC2, S3
- **Authentication**: JWT, OAuth2
- **Monitoring**: Prometheus, Grafana
